# GAME TO AIM LANDING PAGE

Landing Page created by Rahul Nayak for GAME TO AIM CLUB at NIT Rourkela

## Features

- **Responsive Design** - Works seamlessly across desktop, tablet and mobile
- **Interactive Cards** - Flip animation, skew animation and glow effect
- **Smooth Animations** - Smooth FadeIn, Skew and float animations using Framer Motion
- **Neon Theme** - Modern Neon Theme inspired for Gaming

## Tech Stack

- React + Vite (Designing)
- Tailwind CSS (Layout)
- Framer Motion (Animations)
- Lucid React (Icons)
- Lenis (Smooth Scroll)

## Installation and Setup

### Prerequisites

- Node.js (v16 or higher)

### Steps

1. **Clone the repository**

```bash
   git clone https://github.com/RahulNayak6467/Landing-Page
```

2. \*\*Navigate to project directory

```bash
cd Landing-Page
```

3. **Install dependencies**

```bash
   npm install
```

4. **Start development server**

```bash
   npm run dev
```

5. **Open in browser**

```
   http://localhost:5173
```

###Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Create production build
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint (if configured)
